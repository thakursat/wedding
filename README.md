An Indian wedding website template built with Vite, React (JSX) and Tailwind CSS. This project provides a responsive landing site template with components for hero sections, event listings, event detail pages, location, wishes, and simple UI elements to showcase wedding details.

Features

- Clean, responsive layout for wedding sites
- Event listings and event detail pages
- Location and wishes sections
- Lightweight, easy to customize components

Quick start

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

Customize the content and styles in the `src/` directory to match your event.
